## NoXcode macOS App (Xcode project)

This folder holds the SwiftUI app sources. Create a minimal macOS app
project in Xcode and add these files, then add the `NoXcode` package
as a local Swift Package dependency.

Suggested steps:
1. File → New → Project → macOS App
2. Name it `NoXcode` and save the `.xcodeproj` inside this folder.
3. Add the `NoXcode` package (File → Add Packages… → Add Local…)
4. Add the sources in this folder to the app target.
